## 1. Overall Pipeline

Three-stage framework agreed for the project:

1. **Create networks** — fundamental network, passing network, lineup network
2. **Detect mispricing** — use network-derived signals to identify market mispricing
3. **Estimate expected return** — translate mispricing into a tradable signal

## 2. Networks

Multiple network layers to model the team:

- **Lineup network** — how lineups interact with each other (substitution-based, à la Guo et al.)
- **Passing network** — player-to-player ball movement
- **Fundamental network** — built from box-score data (the "inside" network of player stats)
- **Inter-network interactions** — how the layers relate to each other matter, not just each one in isolation

### Network properties to study
- **Resilience to network attacks** — robustness when a node (player/lineup) is removed
- **Community features** — detect sub-structures (e.g., core/connector/peripheral lineups)
- **Non-linear relationships** — model interactions beyond linear assumptions
- **Offense / defense / physical** dimensions captured separately
- **Healthy team hypothesis:** a healthy team should produce a diversified network
- **Min entropy** of the network as a candidate measure
- **WPI estimation** — quantify each part's contribution

## 3. Portfolio / Finance Framing

Treat the NBA betting/trading universe like a portfolio:

- Analogy referenced: **FICO score** — a single composite score derived from many inputs
- Use the network model for **risk management**, not just point prediction
- Core objective: **detect mispricing** in the market relative to model output

## 4. Outcome Measurement

Performance evaluated along **three dimensions**:

1. **Accuracy** — does the model predict outcomes correctly?
2. **Level of mispricing** — how far off is the market vs. the model?
3. **Expected return** — net profitability of acting on the signal
